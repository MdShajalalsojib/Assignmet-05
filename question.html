Question-1: What are the different ways to select an element in the DOM?

Ans:By ID (getElementById), By Class (getElementsByClassName), By Tag Name (getElementsByTagName), By Query Selector (querySelector), By Query Selector All (querySelectorAll), By Query Selector All (querySelectorAll), By Name Attribute (getElementsByName)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:1.innerHTML
Returns or sets the HTML content of an element, including any child elements. It parses the HTML, meaning it processes and renders any HTML tags inside the string.
2.innerText
Returns or sets the visible text inside an element.Ignores hidden elements (display: none).Preserves formatting like line breaks but does not include extra spaces.
